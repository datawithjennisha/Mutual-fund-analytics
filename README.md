# Mutual Fund Analytics Platform

## Project Overview

The Mutual Fund Analytics Platform is an end-to-end data analytics project that analyzes the Indian mutual fund industry using Python, SQL, and Power BI.

The project covers data ingestion, data cleaning, exploratory data analysis (EDA), database design, and interactive dashboard development. It provides insights into NAV trends, AUM growth, SIP inflows, investor demographics, portfolio allocations, and overall mutual fund performance.
